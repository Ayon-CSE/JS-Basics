# Keyboard Event Tracker

A simple web application that displays keyboard information when you press any key.

## Features

- Shows the key name when you press a keyboard key
- Displays the key code value
- Shows the code identifier for each key
- Handles special keys like spacebar with proper display
- Real-time updates as you type

## How to Use

1. Open `index.html` in your web browser
2. Press any key on your keyboard
3. The application displays the key information in a table format

## Technology

- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript for event handling
