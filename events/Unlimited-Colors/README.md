# Unlimited Colors

A simple web application that generates random background colors with interactive controls.

## Features

- Click the Start button to begin generating random colors
- The background changes to a new random color every second
- Click the Stop button to pause the color changes
- Clean, intuitive user interface with responsive button design

## How to Use

1. Open `index.html` in your web browser
2. Click the green Start button to begin
3. Click the red Stop button to stop the color changes

## Technology

- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript for functionality
